# Traveller - 장애인을 위한 여행 플랫폼

## 시작하기

### 백엔드 실행 (포트 3000)
```bash
cd backend
npm install
npm run dev
```

### 프론트엔드 실행 (포트 8080)
```bash
cd frontend
npm install
npm run dev
```

## 기술 스택

- **Frontend**: Vue.js 3, Vuetify 3, Vite
- **Backend**: Node.js, Express.js, MySQL
- **인증**: JWT, bcrypt
- **실시간 통신**: Socket.io

## 주요 기능

- 장애인 맞춤형 여행 계획
- 접근성 정보 제공
- AI 기반 추천 서비스
- 커뮤니티 기능
- 실시간 채팅
